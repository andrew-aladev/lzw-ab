LZWS(3)
=======


NAME
----
lzws_create_destination_buffer_for_compressor - Create destination buffer for compressor.


SYNOPSIS
--------
*#include <lzws/buffer.h>*

----
lzws_result_t lzws_create_destination_buffer_for_compressor(lzws_byte_t** buffer_ptr, size_t* buffer_length_ptr, bool quiet);
----


OPTIONS
-------
*`lzws_byte_t** buffer_ptr`*::
  Pointer to buffer.

*`size_t* buffer_length_ptr`*::
  Pointer to buffer length.

*`bool quiet`*::
  Disable error messages.


RETURN VALUE
------------
Returns *0* if creation was successful.
Returns *LZWS_BUFFER_ALLOCATE_FAILED* when memory allocation was failed.


DESCRIPTION
-----------
This function creates destination buffer for compressor.
You can pass 0 as *buffer_length_ptr* value and it will select default optimal length for performance.
It will write buffer length at *buffer_length_ptr* pointer.

You have to free *buffer_ptr* later.
